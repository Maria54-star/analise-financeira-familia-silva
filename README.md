# 📊 Análise Financeira - Família Silva

Projeto de análise de dados desenvolvido em Python, com visualização gráfica
dos gastos domésticos de uma família carioca de 4 pessoas.

---

## 📌 Sobre o Projeto

Este projeto simula a análise financeira mensal de uma família fictícia
(Família Silva) morando no Rio de Janeiro, com renda combinada de R$ 4.500,00.

O objetivo é visualizar de forma clara como o salário é distribuído
entre gastos fixos e extras ao longo do mês.

---

## 📈 O que o projeto analisa?

- 💧 Gastos fixos: Água, Luz e Internet
- 🛒 Gastos extras: Alimentação, Transporte, Escola dos filhos,
  Lazer e Saúde
- 💼 Salário combinado do casal
- ✅ Saldo final do mês

---

## 📊 Gráficos gerados

| Gráfico | Descrição |
|---|---|
| 🍕 Pizza | Distribuição percentual de todos os gastos |
| 📊 Barras | Comparativo entre salário, gastos e saldo final |

---

- Python 3

1. Acesse [Google Colab](https://colab.research.google.com)

4. Execute e veja os gráficos!
